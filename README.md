# Behujap 
--------
Version 1.0 - Kamis, 26 Feb 2015

by Adam Fikri Maulana

Sinopsis
---------
Aplikasi ini adalah aplikasi yang akan membantu dan memudahkan seseorang dalam mempelajari Huruf Jepang. Aplikasi yang sertai dengan kuis dan penyebutan vocal ini, membuat suasana belajar Huruf Jepang menjadi lebih menyenangkan dan mengasikan.
Dalam aplikasi ini user akan di berikan beberapa menu yaitu *“Huruf Hiragana”*, *“Huruf Katakana”*, *“Latihan Hiragana”*, dan *“Kosa Kata”*. 
Huruf Hiragana, pada menu ini, user akan mempelajari huruf-huruf hiragana.
Huruf Katakana, pada menu ini, user akan mempelajari huruf-huruf katakana. 
Latihan Hiragana adalah latihan yang berguna untuk mengasah daya ingatan user setelah dia mempelajari huruf hiragana tersebut. 
Kosa Kata, pada menu ini terdapat beberapa kosa kata seperti salam dan warna.

Manfaat
-------
* Memudahkan seseorang dalam mempelajari Huruf Hiragana dan Katakana
* Membuat seseorang berminat untuk mempelajari Huruf Jepang
* Membuat suasana belajar huruf jepang menjadi lebih menyenangkan.

Pengguna
--------
Semua orang yang ingin belajar huruf jepang.

Kegiatan
-------
Pada aplikasi ini user akan melakukan kegiatan berikut :
* Mengenal dan memahami huruf hiragana.
* Mengenal dan memahami huruf katakana.
* Menguji kemampuan huruf hiragana.
* Mempelajari kosa kata bahasa Jepang.
